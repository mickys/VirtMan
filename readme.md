# VirtMan
A Libvirt PHP wrapper library for the [Laravel Framework](https://laravel.com/).
_version **0.0.1**_
## Requirements
* PHP >= 7.0
* Libvirt PHP extension
## Installation
```bash
composer require ryanvade/virtman
```
## Examples
### Hello World
```php
$hello = "Hello World";
```
